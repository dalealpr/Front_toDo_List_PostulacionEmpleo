Proyecto

Como parte del proceso de postulación para el cargo de ingeniero backend en Node.js con Express, presento a continuación una aplicación Fullstack To-Do List. Este proyecto ha sido desarrollado como respuesta a la prueba técnica enviada, la cual busca evaluar los conocimientos prácticos en desarrollo backend, organización de proyectos y manejo de rutas, controladores y persistencia de datos. La aplicación permite gestionar tareas de forma eficiente, con funcionalidades como creación, edición, eliminación entre otros.

Además, se ha integrado Socket.IO para actualizar en tiempo real los cambios en las tareas, para probar esta funcionalidad, se recomienda abrir o duplicar la ventana del navegador y observar cómo se reflejan los cambios simultáneamente en la tabla de Tareas.
